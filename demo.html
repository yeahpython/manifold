manifold.js


This is the starter page for a new project on multivariable calculus education.
  • Multivariable calculus is useful for students of many disciplines
  • It's hard to learn
  • It's generally explained in 3D
  • For years, it's been taught on 2D whiteboards, which makes some concepts difficult to convey
  • It's finally possible to teach this stuff in interactive 3D!
Started 2015
